# 🐝 DevBeeZ Landing Page

##  What I Built

- A **pixel-perfect landing page** based on a modern UI design
- Split-screen layout with:
  - Left: Content (heading, description, CTA)
  - Right: Visual section with gradient and illustration
- Clean typography and structured layout for better user experience
- Fully responsive design for desktop and mobile


## 🛠️ Tech Stack Used

### Frontend
- **React.js** (Functional Components)
- **CSS / Tailwind CSS**
- **Flexbox & Grid** for layout
- **React Icons / SVG** for UI elements

### Backend (Project Scope)
- **Node.js**
- **Express.js**

### Database
- **MongoDB**



## 📱 Responsiveness

- Works on all screen sizes
- Mobile-friendly layout (stacked sections)
